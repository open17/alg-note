## 仓库介绍

这是一个 monorepo 仓库，由一下几个部分组成:
- [code](./code)：平时的练习代码
- [template](./template): 算法模板代码
- [docs](./docs/): vitepress站点内容,包含博客与文档,用于更好的浏览内容
- [auto](./auto): 一些自动化,包括自动化目录更新,CF打卡仓库自动推送等
